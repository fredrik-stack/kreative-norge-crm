# Feature: Export

**Status:** planlagt med delvis teknisk grunnlag

Brukeren skal kunne eksportere tenant-avgrensede data basert på søk, filtre, valgt innhold og tilgangsnivå.

Det finnes allerede eksportjobbmodell og grunn-API.

`ADR-005` har besluttet tre planlagte kontaktprodukter: intern arbeidsliste, offentlig katalogeksport fra samme projeksjon som PUBLIC og full relasjonell eksport. Dette er ikke implementert. Felter, tilgangsmatrise, filretensjon, formater, nedlasting og brukerflyt må fortsatt avklares før eksportmotoren ferdigstilles.
