# Feature: Export

**Status:** planlagt med delvis teknisk grunnlag

Brukeren skal kunne eksportere tenant-avgrensede data basert på søk, filtre, valgt innhold og tilgangsnivå.

Det finnes allerede eksportjobbmodell og grunn-API. Før implementering ferdigstilles må vi avklare eksportpakker, felter, relasjoner, personvern, formater, nedlasting og brukerflyt.
